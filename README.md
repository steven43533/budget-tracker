A simple Android budgeting application built in **Java** using **Android Studio**.
Users can register, log in, set up their income and base expenses, and track daily spending through a clean dashboard with charts and summaries.

---

####  Tech Stack

* **Language:** Java
* **Database & Auth:** Firebase (Firestore + Authentication)
* **UI:** Material Design Components
* **Charts:** MPAndroidChart

---

####  Features

* Email/Password authentication (Firebase)
* Profile setup (income, monthly costs)
* Add expenses with categories and notes
* Dashboard with charts showing income vs. spending

---

#### Setup

1. Clone this repository

   ```bash
   git clone https://github.com/steven43533/budget-tracker.git
   ```
2. Open in **Android Studio**
3. Add your `google-services.json` from Firebase into the `/app` folder
4. Sync Gradle and run the project on an emulator or physical device
